FUNCIONALIDADES
===============

AUTENTICACION:
  - Registro de usuarios
  - Login con JWT
  - Roles (admin, cajero)
  - Middleware de protección

DASHBOARD:
  - KPI Total de Ventas
  - KPI Ingresos Totales
  - KPI Ticket Promedio
  - KPI Alertas de Stock
  - Graficos de productos

VENTAS:
  - Seleccion de productos
  - Calculo automatico de impuestos
  - Multiples metodos de pago
  - Historial de transacciones

PRODUCTOS:
  - Listado completo
  - Busqueda y filtrado
  - Crear/Editar (admin)
  - Alertas de stock bajo

INVENTARIO:
  - Vista general de stock
  - Alertas criticas
  - Ajuste de inventario (admin)

REPORTES:
  - Resumen de ventas
  - Exportacion PDF/Excel
  - Desglose por metodo de pago
  - Productos mas vendidos

INICIO RAPIDO
=============

1. Backend (Terminal 1):
   cd backend
   npm install
   cp .env.example .env
   npm run dev
   
2. Frontend (Terminal 2):
   cd frontend
   npm install
   npm run dev

3. Acceder:
   http://localhost:3000
